Mostrar Imagem

Descrição
Letras Góticas é uma aplicação web interativa focada em tipografia animada. As letras surgem, flutuam e explodem no ecrã com uma estética gótica e sombria, criando uma experiência visual dinâmica e imersiva.
A aplicação demonstra técnicas de animação CSS e JavaScript aplicadas à tipografia, explorando movimento, física simplificada e efeitos visuais dramáticos.

Funcionalidades

Letras com animação de flutuação e explosão
Estética tipográfica gótica
Interatividade com o utilizador (input de texto ou gatilhos visuais)
Efeitos de partículas tipográficas


Tecnologias

HTML5
CSS3 (animações e transições)
JavaScript (vanilla)
